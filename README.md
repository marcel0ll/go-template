## Tailwind

```sh
if [ ! -f ./tailwind ]; then
  curl -sLO https://github.com/tailwindlabs/tailwindcss/releases/latest/download/tailwindcss-linux-x64
  mv tailwindcss-linux-x64 tailwind
  chmod u+x tailwind
fi
```
